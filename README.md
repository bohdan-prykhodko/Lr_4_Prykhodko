# Приклад 1

## Мета роботи
- Дослідити фазові траєкторії на площині.
- Визначити положення рівноваги та їх тип.
- Провести лінеаризацію системи.
- Побудувати фазовий портрет та порівняти з аналітичним розв’язанням.

## Умова
Дано систему:
\[
x' = -y, \qquad y' = x - 3x^2.
\]

## Результати
- Знайдено особливі точки: \((0,0)\) та \((\frac13,0)\).
- Лінеаризація показала:
  - у точці \((0,0)\): власні значення \(\lambda = \pm i\) → **центр**, але нестійкий.
  - у точці \((\frac13,0)\): власні значення \(\lambda=\pm1\) → **сідло**.
- Побудовано фазовий портрет, що повторює рисунок у методичці:  
  замкнені майже-колові траєкторії біля центру та сідлові сепаратриси в точці \((\frac13,0)\).
<img width="625" height="624" alt="image" src="https://github.com/user-attachments/assets/807801fc-716d-4ba6-b5ba-dac192683695" />

## Висновки
- Система має два різних за типом положення рівноваги.
- У центрі траєкторії не є замкненими в строгому сенсі — система нестійка.
- У точці \((\frac13,0)\) формується сідло, через яке проходять сепаратриси.
- Побудований чисельний фазовий портрет повністю збігається з результатами з посібника.

# Приклад 2

## Мета роботи
- Знайти всі особливі точки системи.
- Визначити їх характер за допомогою лінеаризації.
- Побудувати фазовий портрет на всій площині.
- Порівняти результат з рисунком з методичного матеріалу.

## Умова
Розглядається система:
\[
\begin{cases}
x' = 2y, \\
y' = 4x - 4x^3.
\end{cases}
\]

## Результати
- Знайдено три положення рівноваги:
  - A(0,0)
  - B(1,0)
  - C(-1,0)
- Лінеаризація показала:
  - у точці A(0,0): \(\lambda = \pm 2\sqrt{2}\) → **сідло**.
  - у точках B(1,0), C(-1,0): \(\lambda=\pm 4i\) → **центри**.
- Фазовий портрет складається з:
  - замкнених еліптичних орбіт навколо центрів;
  - сідлових сепаратрис від (0,0);
  - правильної структури, як у наведеному в посібнику рисунку.
<img width="689" height="701" alt="image" src="https://github.com/user-attachments/assets/cb196829-3470-435a-8623-e42b7853991a" />

## Висновки
- Система має три особливі точки різного типу.
- Центральні точки породжують замкнені періодичні траєкторії.
- Сідлова точка у центрі площини суттєво впливає на глобальну структуру фазового потоку.
- Чисельні результати повністю узгоджуються з теоретичними.
